🦞 OpenClaw 入门助手

一键安装，轻松上手 AI Agent 开发

⚡ 快速安装

npm install -g openclaw

需要 Node.js 18.x 或更高版本

curl -fsSL https://openclaw.ai/install.sh | bash

📋 安装步骤

1

环境检测

运行检测脚本，确认系统满足安装要求

curl -fsSL https://openclaw.ai/check-env.sh | bash
2

安装 OpenClaw

使用 npm 全局安装 OpenClaw CLI

npm install -g openclaw
3

初始化项目

创建你的第一个 AI Agent 项目

openclaw init my-agent cd my-agent
4

配置并启动

配置 API Key 并启动 Gateway 服务

openclaw configure openclaw gateway start

❓ 常见问题

Q: Node.js 版本过低怎么办？

A: 建议升级到 Node.js 18.x 或更高版本。可以使用 nvm 管理 Node 版本：
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20

Q: npm 安装速度慢怎么办？

A: 国内用户建议使用淘宝镜像：
npm config set registry https://registry.npmmirror.com

Q: 没有 API Key 怎么办？

A: OpenClaw 支持多种 AI 提供商。推荐使用：
- OpenAI
- Anthropic Claude
- 阿里云通义千问

Q: 如何在微信/Telegram 上使用？

A: OpenClaw 支持多通道部署。运行 openclaw configure 后选择对应的通道类型，按提示配置即可。
