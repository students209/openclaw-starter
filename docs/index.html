📋 安装步骤

1

环境检测

运行检测脚本，确认系统满足安装要求

curl -fsSL https://raw.githubusercontent.com/students209/openclaw-starter/main/scripts/check-env.sh | bash
2

安装 OpenClaw

使用 npm 全局安装 OpenClaw CLI

npm install -g openclaw
3

初始化项目

创建你的第一个 AI Agent 项目

openclaw init my-agent && cd my-agent
4

配置并启动

配置 API Key 并启动 Gateway 服务

openclaw configure && openclaw gateway start

❓ 常见问题

Q: Node.js 版本过低怎么办？

A: 建议升级到 Node.js 18.x 或更高版本。可以使用 nvm 管理 Node 版本

Q: npm 安装速度慢怎么办？

A: 国内用户建议使用淘宝镜像：npm config set registry https://registry.npmmirror.com

Q: 没有 API Key 怎么办？

A: OpenClaw 支持多种 AI 提供商：OpenAI、Claude、通义千问、Kimi 等
